# Chatty Chatty

This is a friendly chatty

## Installation

Use NodeJS version 20 if possible

`nvm use 20`

Install the dependencies

`npm install`

Create an .env in the root and supply `API_KEY` from OpenAI

```typescript
VITE_API_KEY=YOUR_API_KEY
```

## Usage

Dev

`npm run dev`

Build

`npm run build`
